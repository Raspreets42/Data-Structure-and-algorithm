Queues

A queue is a fundamental data structure that follows the First In, First Out (FIFO) principle, meaning that the first element added to the queue will be the first one to be removed. This data structure is commonly used in scenarios where order matters, such as in scheduling tasks, handling requests, and managing resources.

Basic Operations :
  • Enqueue: Add an element to the end of the queue.
  • Dequeue: Remove and return the element from the front of the queue.
  • Front (Peek): Return the front element without removing it.
  • IsEmpty: Check if the queue is empty.
  • Size: Return the number of elements in the queue.
